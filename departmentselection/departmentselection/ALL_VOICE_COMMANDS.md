## üí° **TIPS FOR BEST RESULTS**

1. **Speak Clearly**: Enunciate each word clearly
2. **Natural Language**: Use natural phrases, not robotic commands
3. **Wait for Feedback**: Listen for voice confirmation before next command
4. **Check Confidence**: Green indicator = good, Red = try again
5. **Try Variations**: If one phrase doesn't work, try another variation
6. **Background Noise**: Minimize background noise for better recognition
